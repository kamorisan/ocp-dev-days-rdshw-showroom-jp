= OpenShift Dev Day

OpenShift Dev Day へようこそ -- Red Hat OpenShift 上で完全な開発者ジャーニーを体験できるハンズオンワークショップです。

== 学べること

このワークショップでは、以下のことを学びます:

* セルフサービス型の内部開発者ポータルとして Red Hat Developer Hub を探索する
* ソフトウェアカタログを使用してアプリケーション、API、ドキュメントを見つける
* Red Hat OpenShift Dev Spaces で事前設定されたクラウド開発環境を起動する
* AI コーディングアシスタントを使用して機能を構築する
* OpenShift Pipelines で自動化された CI/CD パイプラインを通じてコードをプッシュする
* Red Hat OpenShift GitOps (Argo CD) を使用して GitOps で本番環境にデプロイする
* Developer Hub と OpenShift Console からアプリケーションをデバッグおよびモニタリングする

== 対象者

このワークショップは、OpenShift が提供する統合された開発者エクスペリエンス -- コードから本番環境まで -- を理解したい開発者、プラットフォームエンジニア、IT 意思決定者向けに設計されています。

**経験レベル** : 初級から中級。OpenShift の事前経験は必要ありません。

== ワークショップ環境

必要なものはすべて事前にプロビジョニングされています。ラボ環境には以下が含まれます:

* Developer Hub、Dev Spaces、Pipelines、GitOps が事前設定された OpenShift クラスタ
* すべてのワークショップリソースにアクセスできるユーザーアカウント
* 新しい機能で拡張するサンプルアプリケーション (Parasol Insurance)

クレデンシャルはラボインターフェースで利用できます。

== 所要時間

このワークショップの完了には約 **90 分** かかります。

自分のペースで作業できます。
