= OpenShift Dev Day

Welcome to OpenShift Dev Day -- a hands-on workshop where you experience the full developer journey on Red Hat OpenShift.

== What you'll learn

In this workshop, you will:

* Explore Red Hat Developer Hub as a self-service internal developer portal
* Use the software catalog to discover applications, APIs, and documentation
* Launch a preconfigured cloud development environment with Red Hat OpenShift Dev Spaces
* Build a feature using an AI coding assistant
* Push code through an automated CI/CD pipeline with OpenShift Pipelines
* Deploy to production using GitOps with Red Hat OpenShift GitOps (Argo CD)
* Debug and monitor applications from Developer Hub and the OpenShift Console

== Who this is for

This workshop is designed for developers, platform engineers, and IT decision makers who want to understand the cohesive developer experience that OpenShift provides -- from code to production.

**Experience level**: Beginner to intermediate. No prior OpenShift experience is required.

== Workshop environment

Everything you need is already provisioned. Your lab environment includes:

* An OpenShift cluster with Developer Hub, Dev Spaces, Pipelines, and GitOps preconfigured
* A user account with access to all workshop resources
* A sample application (Parasol Insurance) that you will extend with a new feature

Your credentials are available in the lab interface.

== Estimated time

This workshop takes approximately **90 minutes** to complete.

You can work at your own pace.
